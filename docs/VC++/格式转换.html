二、C++ 内置类型转CString

在MFC程序中，可以使用Format方法方便的实现int、float和double等数字类型转换为CString字符串。以下是CString的Format支持的格式说明：

%c 单个字符
%d 十进制整数(int)
%ld 十进制整数(long)
%f 十进制浮点数(float)
%lf 十进制浮点数(double)
%o 八进制数
%s 字符串
%u 无符号十进制数
%x 十六进制数

CString str;
// 1.int->CString
int nNum = 15;
str.Format(_T("%d"),nNum);
// 2.double->CString
